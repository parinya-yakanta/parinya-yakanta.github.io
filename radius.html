Radius And Invoice

รายละเอียดระบบ

เป็นระบบออกใบแจ้งหนี้ให้กับผู้ใช้บริการอินเตอร์เน็ต ซึ่งในระบบจะแบ่งการทำงานเป็น 2 ส่วนหลักคือ ตัวระบบจะมีการตั้งเวลาออกใบแจ้งหนี้ทุกๆ วันที่ 1 ของเดือน และส่วนของแอดมิน

( ส่วนของ Admin )

- จัดการข้อมูลของ raduis user
- จัดการข้อมูล customer
- จัดการสถานะ invoice billing และอื่นๆ ในระบบ

หน้า customer สามารถตั้งการออกบิลแยกได้ และสามารถดู raduis user และ billing รวมถึงระดับ invoice

หน้า raduis user สามารถเข้าไปจัดการข้อมูล รวมถึงปิดการใช้งานได้

หน้า packege สามารถเข้าใปจัดการข้อมูลได้

หน้า billing สามารถเรียกดูเป็นรายเดือนได้ และสามารถเข้าไปอัพเดตสถานะ รวมถึงเพิ่มรายการข้างในได้

หน้าพรีวิว PDF สถานะจะหายไปเมื่ออัพเดตเป็นชำระเงินแล้ว

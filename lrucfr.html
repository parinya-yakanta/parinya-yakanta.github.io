ระบบลงทะเบียน และส่งผลงานเข้าร่วมการประชุมวิชาการระดับชาติ

รายละเอียดระบบ

เป็นระบบที่ให้นักวิจัยลงทะเบียนเพื่อส่งผลงานเข้าร่วมการประชุมวิชาการ ซึ่งระบบแบ่งออกเป็น 2 ส่วนหลักๆ ได้แก่ User / Admin

( ส่วนของ User )

- ลงทะเบียน
- ยืนยันตัวตนผ่านอีเมล
- เข้าระบบ
- ส่งผลงาน : แนบไฟล์ pdf, word, ไฟล์หลักฐานการชำระเงิน
- ดูสถานะการดำเนินการของบทความ
- ส่งบทความแก้ไข
- แก้ไขข้อมูลส่วนตัว

หน้าหลัก

- หน้าหลักของ user จะแสดงบทความของตนเองท้งหมด

หน้าโปรไฟล์

- หน้าโปรไฟล์ของผู้ใช้สามารถเข้าไปแก้ไขข้อมูลได้บางส่วน

หน้าส่งผลงาน

- หน้าส่งผลงานเข้าระบบของ user

- หน้าส่งผลงานเข้าระบบของ user สามารถแนบไฟล์เข้าระบบได้

หน้าส่งบทความแก้ไข

( ส่วนของ Admin )

- จัดการสถานะบทความ
- จัดการข้อมูล user
- ส่งไฟล์บทความ : แนบไฟล์ pdf, word, ไฟล์หลักฐานการชำระเงิน
- เพิ่ม proceeding
- ดูรายงานอื่นๆ

หน้าเว็บไซต์
